# Quiz Tự Làm

Ứng dụng web quiz tự làm với Next.js.

## Bắt Đầu

```bash
npm install
npm run dev
```

## Cấu Trúc

```
src/
├── app/              # Next.js App Router
├── components/       # UI Components
├── hooks/           # Custom React Hooks
├── lib/             # Utility functions
└── types/           # TypeScript types
```
